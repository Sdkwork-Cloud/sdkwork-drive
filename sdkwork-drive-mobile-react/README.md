# sdkwork-drive-mobile-react

English is the default language for this README. A Chinese version is provided below.

## Overview

`sdkwork-drive-mobile-react` is the reserved workspace for the future React-based mobile client of SDKWork Drive.

At the moment, this directory is intentionally minimal and acts as a placeholder in the repository structure. The current product implementation focus remains on [`../sdkwork-drive-pc-react`](../sdkwork-drive-pc-react).

## Current Status

- Repository placeholder only
- No production mobile implementation has been landed yet
- Reserved for a future React mobile architecture aligned with the Drive product system

## Planned Direction

The planned mobile React client is expected to inherit:

- Shared Drive domain capability where feasible
- Auth and user model alignment with the main Drive product
- Consistent release and SDK strategy with the broader repository
- Mobile-first interaction patterns rather than direct desktop layout reuse

## Notes

Until implementation starts, use [`../sdkwork-drive-pc-react`](../sdkwork-drive-pc-react) as the active project workspace.
