# Developer Guide

Local development for SDKWork Drive starts from the repository root:

- `pnpm dev` — browser standalone workflow with PostgreSQL defaults
- `pnpm dev:desktop` — Tauri desktop host
- `pnpm test` / `pnpm verify` — validation before merge

PC application root: `apps/sdkwork-drive-pc/`.

See `../../../sdkwork-specs/DOCUMENTATION_SPEC.md` section 2.1.
