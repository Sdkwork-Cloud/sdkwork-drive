# sdkwork-drive-mobile-flutter

English is the default language for this README. A Chinese version is provided below.

## Overview

`sdkwork-drive-mobile-flutter` is the reserved workspace for the future Flutter-based mobile client of SDKWork Drive.

This directory is currently a placeholder and does not yet contain an implemented mobile application. The active product implementation is in [`../sdkwork-drive-pc-react`](../sdkwork-drive-pc-react).

## Current Status

- Placeholder only
- No Flutter mobile app has been implemented yet
- Reserved for a future mobile client when Flutter becomes the chosen delivery path

## Planned Direction

The future Flutter workspace is expected to focus on:

- Mobile-native Drive experience
- Shared product language with the desktop/web Drive product
- Compatibility with repository-wide auth, user, and SDK boundaries
- A mobile release strategy that fits Flutter packaging and distribution

## Notes

Until this workspace is implemented, all active Drive product development is centered on [`../sdkwork-drive-pc-react`](../sdkwork-drive-pc-react).
