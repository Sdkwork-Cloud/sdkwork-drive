# Developer Guide

```bash
pnpm dev          # browser renderer (127.0.0.1:5183)
pnpm dev:desktop  # Tauri shell
pnpm test
pnpm typecheck
```

Start Drive services from repository root (`pnpm dev`) before opening the renderer.

See [TECH_ARCHITECTURE.md](../architecture/tech/TECH_ARCHITECTURE.md) and [../../../sdkwork-specs/DOCUMENTATION_SPEC.md](../../../sdkwork-specs/DOCUMENTATION_SPEC.md) section 2.1.
